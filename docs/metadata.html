Chapter 3 Metadata

3.1 Date formats

Use ISO 8601 formatted dates YYYYMMDD
This date format enables easy chronological sorting

For example, 5 Jan 2012 would be 20120105

3.2 Use the underscore _ to seperate elements in file names

This refers to the characters you will use to separate each piece of metadata in the file name. Many computer systems cannot handle spaces in file names, so NEVER use spaces!
